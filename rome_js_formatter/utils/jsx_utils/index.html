This is a very special situation where we’re returning a JsxElement from an arrow function that’s passed as an argument to a function, which is itself inside a JSX expression child.

is_meaningful_jsx_text

Meaningful JSX text is defined to be text that has either non-whitespace characters, or does not contain a newline. Whitespace is defined as ASCII whitespace.
